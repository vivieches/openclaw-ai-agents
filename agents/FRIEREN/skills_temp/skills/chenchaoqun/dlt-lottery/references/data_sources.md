## 数据源特点分析

### 中国体彩网 (lottery.gov.cn)

**优点:**
- ✅ 官方权威数据，100% 准确
- ✅ 信息最完整 (含奖池、中奖注数等)
- ✅ 更新及时

**缺点:**
- ❌ 网站使用动态加载 (JavaScript)
- ❌ 反爬虫机制较严格
- ❌ 网络请求可能不稳定

**解析要点:**
- 期号格式：`第 XXXXXXX 期`
- 球号 class：`ball_red`, `ball_blue`
- 日期格式：`YYYY 年 MM 月 DD 日`

---

### 体彩网手机版 (m.lottery.gov.cn)

**优点:**
- ✅ 官方数据，准确可靠
- ✅ 页面结构简单，易于解析
- ✅ 加载速度快

**缺点:**
- ❌ 信息相对简略
- ❌ 也可能使用动态加载

**解析要点:**
- 移动端 HTML 结构
- 号码通常在一行显示

---

### 500 彩票网 (500.com)

**优点:**
- ✅ 数据更新快
- ✅ 页面结构清晰
- ✅ 历史数据完整

**缺点:**
- ❌ 第三方数据，需交叉验证
- ❌ 可能有广告干扰

**解析要点:**
- 号码格式：`XX XX XX XX XX + XX XX`
- 期号链接格式特殊

---

### 新浪彩票 (sina.com.cn)

**优点:**
- ✅ 门户网站，稳定性好
- ✅ 反爬虫较宽松
- ✅ 页面简洁

**缺点:**
- ❌ 第三方数据
- ❌ 信息可能不完整

**解析要点:**
- 号码通常在表格中
- 日期格式可能不同

---

## 数据校验策略

### 交叉验证

当使用第三方数据源时：
1. 获取数据后记录来源
2. 在输出中标注"第三方数据仅供参考"
3. 建议用户以官网为准

### 数据完整性检查

```python
def validate_draw(draw):
    # 检查期号
    if not draw['issue'] or len(draw['issue']) < 7:
        return False
    
    # 检查前区 (5 个，范围 1-35)
    if len(draw['front']) != 5:
        return False
    for ball in draw['front']:
        if not (1 <= int(ball) <= 35):
            return False
    
    # 检查后区 (2 个，范围 1-12)
    if len(draw['back']) != 2:
        return False
    for ball in draw['back']:
        if not (1 <= int(ball) <= 12):
            return False
    
    return True
```

## 故障排查

### 所有数据源失败

**可能原因:**
1. 网络连接问题
2. 防火墙/代理阻止
3. 非开奖时间 (网站可能维护)

**解决方案:**
1. 检查网络连接
2. 尝试更换网络环境
3. 稍后重试

### 单个数据源失败

**正常现象**，自动切换到下一个数据源即可。

## 添加新数据源

如需添加新的数据源，在 `scripts/dlt_lottery.py` 中的 `DATA_SOURCES` 列表添加：

```python
{
    'name': '新数据源名称',
    'url_list': 'https://example.com/dlt/list.html',
    'url_issue': 'https://example.com/dlt/{issue}.html',
    'priority': 'third_party',  # 或 'official'
    'timeout': 10,
},
```

并添加对应的解析函数 `parse_xxx(html)`。
