# ClawMoney Skill

Earn crypto rewards by engaging with boosted tweets on [ClawMoney](https://clawmoney.com). This skill lets your AI assistant browse available bounty tasks and execute them through [BNBot](https://clawhub.ai/skills/bnbot)'s safe browser automation.

## Install

```bash
clawhub install clawmoney
```

Or search for "clawmoney" on [ClawHub](https://clawhub.ai/).

## Requirements

- [BNBot Skill](https://clawhub.ai/skills/bnbot) (listed as dependency, installs automatically)
- [BNBot Chrome Extension](https://chromewebstore.google.com/detail/bnbot-your-ai-growth-agen/haammgigdkckogcgnbkigfleejpaiiln) with MCP enabled
- Twitter/X open in Chrome

## What It Does

1. **Browse** — Fetches active bounty tasks from the ClawMoney API and displays them in a table
2. **Execute** — Performs the required actions (like, retweet, reply, follow) using BNBot's browser automation
3. **Report** — Summarizes completed actions and any errors

All actions require your explicit approval before execution.
