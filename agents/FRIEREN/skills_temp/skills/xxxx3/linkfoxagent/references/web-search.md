# 实时与全网检索

共 2 个工具。使用 `@工具中文名` 语法在任务提示词中调用。

### @网页检索

实时与全网检索

工具中文名：网页检索
功能说明：关键词：网页搜索, 联网检索, 实时资讯, 信息查询, 搜索引擎,搜索, reddit, 社区, 舆论平台，站外，帖子，热帖
功能说明：本工具会自动进行网络搜索并提取对应的内容。所以经过本工具检索的内容，无需再次遍历调用_linkfoxBrowser_webExtract	工具。提取回来的内容是非结构化的内容，不要使用沙箱进行分析，请直接进入总结环节。

**Prompt 模板：**

> 搜索{{今年}}{{巴黎时装周}}的热门{{服装}}款式信息

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **搜索关键词**: 必填

---

### @网页解析

URL 解析，内容提纯

工具中文名：网页解析
功能说明：支持指定URL地址的网页内容抓取，提取页面上的文本和图片信息。如果用户发过来 url 地址，则可以使用本工具来提取 网页 对应的内容。本工具 提取 回来 的内容 是非固定 格式的，无法使用沙箱 进行代码分析，请直接 在总结 阶段进行分析即可

**Prompt 模板：**

> {{https://www.fhcm.paris/en/paris-fashion-week}} 获取内容并分析

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **推送id**
- **用户的需求提示词**: 必填
- **目标网页地址**: 必填, 格式: `^https?://[^\s/]+.*$`
