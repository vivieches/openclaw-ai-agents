# Template for New Skills

This is a starter template following the Nima Skill Creator framework.

## Directory Structure

```
template-skill/
├── SKILL.md              # Required
├── scripts/              # Optional
├── references/           # Optional
└── assets/               # Optional
```

## SKILL.md Template

```markdown
---
name: skill-name-here
description: Clear description of what the skill does and when to use it
---

# Skill Name

## Overview
Brief overview of what this skill does.

## Core Features
- Feature 1
- Feature 2
- Feature 3

## How to Use
Basic usage instructions.

## Example
Example of the skill in action.

## Best Practices
Any important tips or guidelines.
```

## Next Steps

1. Copy this template
2. Rename to your skill name
3. Edit SKILL.md with your content
4. Add any needed scripts, references, or assets
5. Test your skill
6. Package and distribute
