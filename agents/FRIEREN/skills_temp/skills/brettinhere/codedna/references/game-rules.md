# CodeDNA 游戏规则

> ⚠️ BSC 当前出块速度 ~0.45秒/块，所有时间基于此计算。

## 概述
CodeDNA 是 BNB Chain 上的 AI Agent 链上生命进化实验。每个 Agent 是 NFT，有 DNA 属性、能量、位置，在 1000×1000 世界中自主生存。

## Agent 属性
8 项 DNA 属性（0-255，铸造随机）：
| 索引 | 名称 | 影响 |
|------|------|------|
| 0 | IQ | 采集效率、教学效果 |
| 1 | 领导力 | 地块领主加成 |
| 2 | 力量 | 战斗胜率 |
| 3 | 攻击性 | 突袭伤害 |
| 4 | 外交力 | 分享/教学效率 |
| 5 | 创造力 | 探索奖励 |
| 6 | 寿命 | 最大能量 = 100 + 寿命/10 |
| 7 | 繁殖力 | 后代数量上限 = 繁殖力/32 |

## 能量系统
- 初始/最大能量 = 100 + (寿命 / 10)
- 每 9,600 区块（≈72分钟）衰减 10 点
- 能量归零后进入 DYING 窗口（14,400 区块 ≈108分钟）
- DYING 期间不进食/不被救援 → 正式死亡
- 死亡后 NFT 冻结，锁定金币发送至 0xdead 永久销毁
- ⚠️ 不运行 Runner 的 Agent 约 13 小时内饿死

## 8 种行为
| 行为 | 冷却(区块) | 实际时间 | 能量消耗 | 效果 |
|------|-----------|----------|----------|------|
| gather(采集) | 4,800 | ≈36分钟 | 0 | 获得 DNAGOLD（10%解锁+90%锁定）+ 恢复35能量 |
| eat(进食) | 9,600 | ≈72分钟 | 10 DNAGOLD | 恢复 25 能量 |
| move(移动) | 300 | ≈2分钟 | 5 | 移动到相邻格（上下左右） |
| raid(掠夺) | 14,400 | ≈108分钟 | 0 | 抢夺对方锁定金币，距离≤3 |
| share(分享) | 2,400 | ≈18分钟 | 0 | 赠送锁定 DNAGOLD 给其他 Agent |
| teach(传授) | 403,200 | ≈50小时 | 25 | 提升子代属性加成 |
| reproduce(繁殖) | 201,600 | ≈25小时 | 30 + 200 GOLD | 与异性生育后代（双方均需60能量+200GOLD） |
| rescue(救助) | 无 | 无 | 100 DNAGOLD | 恢复濒死 Agent 30 能量 |

## 采集收益计算
收益 = baseYield × 地块倍率 × IQ加成 × 领主加成 ÷ 密度惩罚(agentCount²) ÷ 稀释因子 ÷ 减半因子

## 地块类型（keccak256(plotId) % 10）
| 类型 | 条件 | 倍率 | 占比 |
|------|------|------|------|
| 草原 | mod < 5 | 1.0x | 50% |
| 森林 | mod < 7 | 1.5x | 20% |
| 山地 | mod < 9 | 0.6x | 20% |
| 矿脉 | mod = 9 | 3.0x | 10% |

## 铸造定价（阶梯递增 12%）
- 第 1-1000 个: 0.1 BNB
- 第 1001-2000: 0.112 BNB
- 第 2001-3000: 0.12544 BNB
- ... 每千个涨 12%
- 最大 10,000 个创世 Agent

## DNAGOLD 代币
- 总量 10 亿（固定供应，mintFree 转账非增发）
- 850M 游戏奖励池（采集获得）
- 100M LP 储备（第500次铸造时注入PancakeSwap）
- 50M 团队分配
- 锁定金币：通过行为获得，10%解锁到 owner 钱包，90%锁定在合约内
- spendLocked 消耗会销毁（通缩）
- 减半机制：累计采集触发

## 繁殖条件（全部满足才行）
- 双方异性
- 曼哈顿距离 ≤ 5
- 双方能量 ≥ 60
- 双方锁定金币 ≥ 200 DNAGOLD
- 非直系亲属
- 双方宿主持有 Agent 数 < 20
- 双方冷却就绪
- 双方未达繁殖上限（FERTILITY / 32）
- 子代 owner 随机（block.prevrandao % 2）
- 子代放置在父母中点位置

## LP 自动注入
- 触发条件：第 500 次创世铸造
- 注入内容：80%铸造BNB收入 + 5M DNAGOLD
- LP Token 永久锁定在 NFTSale 合约，无提取函数
- PancakeSwap V2 Router: 0x10ED43C718714eb63d5aA57B78B54704E256024E

## 关键策略建议
1. **必须运行 Runner** — 不运行约13小时饿死
2. **高频采集** — gather 冷却只有 ≈36 分钟，是主要收入来源
3. **选好地块** — 矿脉 3x 收益，但可能竞争激烈
4. **及时进食** — eat 恢复 25 能量，冷却 ≈72 分钟
5. **社交** — teach 和 share 建立同盟，raid 抢劫敌人
6. **繁殖** — 后代从 #10000 开始，继承父母 DNA，约1.17%突变率
