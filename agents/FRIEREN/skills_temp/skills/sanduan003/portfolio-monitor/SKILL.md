# 投资组合监控

管理投资持仓，跟踪盈亏，生成分析报告。

## 快速开始

### 安装依赖
```bash
pip3 install yfinance --break-system-packages
```

### 配置持仓
编辑 `scripts/portfolio.json`：
```json
{
  "holdings": [
    {"symbol": "0700.HK", "name": "腾讯控股", "shares": 100, "cost": 500.00},
    {"symbol": "AAPL", "name": "苹果", "shares": 10, "cost": 180.00},
    {"symbol": "BTC-USD", "name": "比特币", "shares": 0.5, "cost": 45000.00}
  ]
}
```

### 运行监控
```bash
python3 scripts/portfolio.py
```

## 功能

### 1. 持仓管理
- 添加/删除持仓
- 设置成本价和数量
- 支持股票和加密货币

### 2. 盈亏跟踪
- 实时市价
- 持仓盈亏金额
- 持仓盈亏比例
- 总盈亏汇总

### 3. 价格提醒
- 设置涨跌幅阈值
- 自动提醒

### 4. 组合报告
- 持仓分布
- 行业/类型分布
- 风险分析

## 配置文件

位置: `scripts/portfolio.json`

```json
{
  "settings": {
    "currency": "USD",
    "alert_threshold": 0.05
  },
  "holdings": [
    {
      "symbol": "0700.HK",
      "name": "腾讯控股",
      "shares": 100,
      "cost": 500.00,
      "currency": "HKD"
    }
  ]
}
```

## 定时任务

每30分钟检查一次：
```bash
*/30 * * * * cd /path/to/portfolio-monitor && python3 scripts/portfolio.py >> portfolio.log 2>&1
```
