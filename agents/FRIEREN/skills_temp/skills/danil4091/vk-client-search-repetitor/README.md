# 🎓 VK Client Search (Repetitor Edition)
**Скилл для OpenClaw: Автоматический поиск учеников по математике**

Этот навык превращает вашего AI-агента в неутомимого менеджера по продажам. Он каждые 3 часа сканирует целевые группы ВКонтакте, находит живые запросы от родителей и фильтрует их по приоритету (Онлайн/Офлайн).

## 🚀 Быстрый старт

1.  **Установка**: Положите папку `vk-client-search-repetitor` в директорию `/skills/` вашего OpenClaw.
2.  **Настройка**: Откройте файл `config.json` и заполните два поля:
    * `access_token`: Вставьте ваш сервисный ключ доступа VK (инструкция ниже).
    * `target_groups`: Впишите ссылки на 10-15 групп, где сидит ваша аудитория.
3.  **Запуск**: Скажите агенту: *"Запусти мониторинг клиентов в ВК"*.

## 🔑 Где взять Access Token? (Для пользователя)

Вам не нужно быть программистом. Это бесплатно и занимает 2 минуты:
1. Перейдите в [VK Developers](https://vk.com/apps?act=manage).
2. Нажмите **"Создать приложение"** (выберите "Доверенное приложение").
3. Перейдите в раздел **"Настройки"**.
4. Скопируйте **"Сервисный ключ доступа"**. Это и есть ваш `access_token`. 
*Никому не передавайте этот ключ!*

## 📊 Что вы получите на выходе?

Агент будет вести таблицу `leads_math.csv`, разделяя заявки:
* 🔥 **ВЫСОКИЙ ПРИОРИТЕТ**: Родители прямо пишут "Онлайн", "Zoom", "Skype".
* ⚡ **СРЕДНИЙ ПРИОРИТЕТ**: Общий запрос "Ищу репетитора".

Раз в 3 часа вы будете получать краткую сводку в чат: сколько новых "лидов" найдено и ссылки на них.
