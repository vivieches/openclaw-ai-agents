# Immigration & Visa Compliance Agent

US employment-based immigration compliance for businesses. Covers visa category selection (H-1B, L-1, O-1, TN, E-2), prevailing wage and LCA requirements, I-9 audit readiness, PERM/green card employer obligations, H-1B lottery strategy, and worksite enforcement trends.

Built for HR teams, immigration paralegals, in-house counsel, and business owners sponsoring foreign workers.

## What's Inside

- **8 visa categories** with costs, timelines, and use cases
- **H-1B lottery strategy** including cap-exempt paths and backup options
- **Prevailing wage compliance** — wage levels, LCA posting, public access file
- **I-9 audit checklist** — ICE fine schedule, retention rules, E-Verify
- **PERM recruitment** — mandatory ads, recruitment report, 5-year retention
- **Priority date strategy** — EB-1/2/3 backlogs by country, alternative paths
- **Compliance calendar** — month-by-month immigration actions
- **Enforcement trends** — ICE audit volumes, state-level rules, remote work impacts
