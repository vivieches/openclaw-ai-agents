# PEER REVIEW PROTOCOL — Squad VY (Custo Zero)

## 🎯 Objetivo
Garantir qualidade sem desperdício de tokens.

## 🔄 Fluxo Compacto
1. **PULSE** centraliza a task.
2. **ESPECIALISTA** entrega (ex: ECHO).
3. **CRÍTICO** ataca (3 pontos, máx 300 tokens).
4. **MIRROR** dá o veredito final (**APROVADO/REJEITADO**) baseado em fatos.

## 📜 Regras de Ouro
- **Veredito Vlock**: Máximo de **200 tokens**.
- **Sem Ruído**: Proibido elogios ou introduções longas.
- **Cache**: Resultados finais salvos no Redis `squad:review:verdict`.
